# DSA-1-2

Data Structures and Algorithms implementations in C.

## Folder Structure

- **LinkedList/** - Linked List operations (insert, delete, node creation)
- **Stack/** - Stack implementation using dynamic array
- **Queue/** - Queue and Circular Queue
- **Recursion/** - Recursion examples (factorial, fibonacci, sum, etc.)
- **Searching/** - Linear and Binary Search algorithms
- **DynamicMemory/** - Dynamic memory allocation examples
- **Fibonacci/** - Fibonacci implementations

## Languages
C
